# Hotels Data Analysis Project

## Overview

This project focuses on analyzing hotel booking data to generate business insights related to occupancy, revenue, booking trends, room utilization, and customer behavior. The dataset consists of dimension and fact tables following a star schema commonly used in data warehousing and business intelligence projects.

## Dataset Files

### 1. `dim_date.csv`

### 2. `dim_hotels.csv`

### 3. `dim_rooms.csv`

### 4. `fact_aggregated_bookings.csv`

### 5. `fact_bookings.csv`

## Project Objectives

The main objectives of this project are:

* Analyze hotel occupancy trends.
* Measure booking performance across properties.
* Calculate revenue metrics and growth trends.
* Identify high-performing hotels and room categories.
* Evaluate cancellation and no-show impacts.
* Monitor customer booking behavior over time.

### Booking Metrics

### Time-Based Metrics

* Monthly Revenue Trend
* Weekly Occupancy Trend
* Seasonal Booking Analysis
